
`python -m <module/package name>`  executes the module or `__main__.py` of a package.

`pyplot.subplots()` - returns two objects `fig,ax` 

*fig* - overall canvas that holds one or more plots.
*ax* - single *Axes* object or *numpy array of Axes*

*Axes* - single plotting area
*Axis* -  x/y axis inside an Axes
